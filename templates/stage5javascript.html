Lesson 1: Meet Javascript

Javascript

The programming language of the web. Javascript interacts directly with your HTML.

Adding Javascript

There is two ways you can add javascript to your html similar to how you add CSS. The first way is to write your javascript inside of a script tag inside of your html. The other way is to create a separate file with the extension .js

Javascript Variables

var variableName = someValue

var declares a new variables for all data types. the main difference between python and javascript is the keyword “var"

Loading Javascript into HTML

Javascript files load within HTML in the order that they are placed within the HTML code.

Truthy vs. Falsy

Evaluate to True:

  • True
  • non-zero numbers
  • "strings"
  • objects
  • arrays
  • functions


Evaluate to False:
  • False
  • 0
  • "" empty string
  • undefined = variable does not exist
  • null = variable exists but no value (usually a placeholder)
  • NaN = not a number

Arrays

Arrays in javascript are just like lists in python and behave the same way.

var myArray = [item_1, item_n]

Array Manipulation Solution In My Own Words:

The goal, if given an array of numbers can you increase the value of the last number in the array by one.

function incrementLastArrayElement(_array)
{
var newArray = [];
newArray = _array.slice(0);
var lastNumber = newArray.pop();
newArray.push(lastNumber + 1);
return newArray;
}

  1. The array in question is passed into the function incrementLastArrayElement. That array technically could be one number or 100 all we care about is whatever is last.
  2. Since we will be adjusting that array, first we will copy it into a new variable using .slice(). .slice() will extract the elements from the beginning up to the end. In this example we want to extract the whole array and store it into a new array

    var newArray = _array.slice(0);
  3. Now we need to increase our lasNumber by 1 and insert it back into the array using .push().

    newArray.push(lastNumber + 1);

Objects

- Javascript does not have classes only objects.

- Object literal notation includes var objectName = { };

var bio = {
"name" : "james",
"age" : 25,
"skills" : skills,
};

- We can add new properties to an object using dot notation. In the example above we could add things to bio using dot notation ex:

bio.city = “Columbia”;
bio.email = “ktryber@gmail.com”;

- You can then append them to the page using dot notation and append.

- 3 ways of defining objects:

  • var myObject = { }; object literal notation
  • myObject.property = someValue; dot notation
  • myObject[“property”] = someValue; bracket notation

What is JSON?

- Javascript Object Notation

- JSON is a simple and popular format for storing and transferring nested or hierarchal data. Most languages have libraries capable of parsing and writing JSON.
